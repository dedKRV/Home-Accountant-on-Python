Описание идеи
----------------------
Проект "Домашний бухгалтер" представляет собой комплексное приложение для полноценного управления личными финансами, разработанное на языке Python с использованием библиотеки PyQt6 для создания графического интерфейса.

Основная цель проекта - предоставить пользователям удобный и функциональный инструмент для ежедневного учета финансовых операций, анализа расходов и доходов, планирования бюджета и формирования отчетности. Приложение ориентировано на широкий круг пользователей - от студентов и молодых семей до предпринимателей, ведущих учет личных финансов.

Ключевой особенностью системы является разделение основного счета и накопительного вклада, что позволяет пользователям эффективно управлять как текущими расходами, так и долгосрочными накоплениями. Интуитивно понятный интерфейс и автоматизированные расчеты делают процесс ведения домашней бухгалтерии простым и доступным даже для пользователей без специальной финансовой подготовки.

Технические решения:
----------------------

Двухуровневая система хранения данных - комбинация CSV-файлов для пользовательского доступа и SQLite базы для системных нужд обеспечивает как простоту использования, так и надежность хранения.

Система валидации - проверка корректности вводимых данных на уровне интерфейса с немедленным уведомлением пользователя об ошибках.

Механизм слияния отчетов - уникальная функция объединения данных из нескольких периодов (3 месяца для квартальных отчетов, 12 месяцев для годовых) с автоматической сортировкой по датам.

Визуализация данных - собственная реализация построения диаграмм с помощью QPainter без использования сторонних библиотек.

Система восстановления сеанса - автоматическое сохранение и восстановление пути к последней открытой таблице, позволяющее пользователю быстро продолжить работу с того места, где он остановился.

Технологический стек:
----------------------

Язык программирования: Python 3.8+
